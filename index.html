  • 대구의 창

    본 프로젝트는 대구의 다양한 관광 명소를 소개하여
    대구를 방문하는 관광객들에게 유용한 정보를 제공하는 것을 목표로 합니다.

    1. 대구의 주요 관광지를 테마별로 소개합니다.
    2. 주요 관광지별 상세정보를 보여줍니다.
    3. 관심있는 관광지는 찜하고 사용자가 다시 볼 수 있습니다.
    4. 내가 가고 싶은 관광지만을 클릭하여 나만의 코스를 만들 수 있습니다.

    사용 툴 : Figma, Photoshop, Javascript, Html, Css
    참여도 : 디자인 100%, 웹퍼블싱 : 80%

  • 별책부록

    별책부록은 대구의 독립서점 문화를 활성화하고 스탬프를
    이용해 방문을 유도하는 앱입니다.

    1. 대구 지역 독립서점의 위치를 제공합니다.
    2. GPS인증을 통해 QR코드를 기반으로 스탬프 적립을 할 수 있습니다.
    3. 방문과 책 구매로 스탬프를 적립할 수 있고 각각의 할인 쿠폰을 받을 수 있습니다.
    4. 다시 시작할 수 있어 지속적인 참여와 방문을 유도합니다.
    5. 서점 방문 후기나 구매 도서 소개 및 추천글을 적을 수 있고 다른 사용자들의 글들도 열람할 수 있습니다

    사용 툴 : Figma, Photoshop, Javascript, Html, Css, React
    참여도 : 웹퍼블리싱 100%, 프론트앤드 70%
